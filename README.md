# 🎬 우리 반 애니메이션 스튜디오

초등학교 5학년 학생들이 **애니메이션을 배우고 → 계획하고 → 사진으로 찍어 영상을 만들고 → 친구들과 보드에서 함께 보며 하트·댓글·동료평가**를 하는 웹앱입니다.

- 💚 **전부 선생님의 구글 계정 안에서** 동작합니다. (Firebase·결제카드·외부 가입 전혀 없음)
- 🔗 학생들은 **링크 하나만** 열면 됩니다. (로그인 불필요)
- 📁 데이터는 선생님 구글 드라이브의 **시트 1개 + 폴더 1개**에 자동 저장됩니다.

---

## 📦 무엇으로 만들어졌나요?

| 역할 | 사용 도구 |
|------|-----------|
| 학생 요청 처리(서버) | 구글 앱스 스크립트 (`Code.gs`) |
| 화면 | `Index.html` + `Style.html` + `App.html` + `Studio.html` |
| 명단·하트·댓글·평가 저장 | 구글 시트(자동 생성: **"애니메이션 스튜디오 데이터"**) |
| 영상(사진 컷) 저장 | 구글 드라이브 폴더(자동 생성: **"애니메이션 스튜디오 작품"**) |

> 시트와 폴더는 **처음 한 번 자동으로 만들어집니다.** 선생님이 미리 만들 필요가 없어요.

## 🚀 설치하기 (선생님, 딱 한 번만 하면 됩니다)

### 1단계 — 앱스 스크립트 프로젝트 만들기
1. 크롬에서 **https://script.google.com** 접속 → 구글 로그인(학교 계정 권장)
2. 왼쪽 위 **`+ 새 프로젝트`** 클릭

### 2단계 — 코드 5개 붙여넣기
앱스 스크립트 편집기 왼쪽 **파일(📄) 영역**에서 파일을 만들고, 이 폴더(`apps-script`)의 같은 이름 파일 내용을 그대로 복사해 붙여넣습니다.

| 편집기에서 만들 파일 | 종류 | 붙여넣을 내용 |
|--------------------|------|--------------|
| 기본 `Code.gs` (이미 있음) | 스크립트 | `apps-script/Code.gs` 전체 |
| `Index` | **HTML** | `apps-script/Index.html` 전체 |
| `Style` | **HTML** | `apps-script/Style.html` 전체 |
| `App` | **HTML** | `apps-script/App.html` 전체 |
| `Studio` | **HTML** | `apps-script/Studio.html` 전체 |

> 📌 **중요**
> - 파일 추가는 파일 영역의 **`+` → `HTML`** 을 누르고 이름을 입력합니다. (`.html` 은 빼고 `Index`, `Style`, `App`, `Studio` 라고만 입력)
> - HTML 파일 이름은 **대소문자까지 똑같이** `Index`, `Style`, `App`, `Studio` 여야 합니다.
> - `Code.gs` 파일의 이름은 아무래도 괜찮습니다. (기본 이름 그대로 두세요)
> - 다 붙여넣었으면 **저장(💾 또는 Ctrl+S)**

### 3단계 — 웹앱으로 배포하기
1. 오른쪽 위 **`배포` → `새 배포`**
2. 톱니바퀴(유형 선택) → **`웹 앱`** 선택
3. 설정:
   - **설명**: 애니메이션 스튜디오 (아무거나)
   - **실행 계정**: **나(선생님)**
   - **액세스 권한**: **모든 사용자** (또는 학교 도메인을 쓰면 *“○○학교 사용자”*)
4. **`배포`** 클릭 → 처음이면 **권한 승인** 창이 뜹니다.
   - `액세스 승인` → 본인 구글 계정 선택
   - *“이 앱은 Google에서 확인하지 않았습니다”* 경고가 나오면 → **`고급`** → **`(프로젝트 이름)(으)로 이동`** → **`허용`**
5. 나오는 **웹 앱 URL**(`.../exec` 로 끝남)을 **복사**합니다. → 이게 학생들에게 줄 주소예요!

### 4단계 — 학생 명단 등록
1. 복사한 웹앱 URL을 크롬에서 엽니다.
2. 오른쪽 위 **⚙️** → 비밀번호 **`1234`** 입력 (첫 비밀번호)
3. **학생 명단**에 한 줄에 한 명씩 입력 → **`명단 저장`**
   ```
   1,김하늘
   2,이보람
   3,박솔
   ```
   - `번호,이름` 형식이 좋지만, 그냥 이름만 한 줄씩 적어도 됩니다.
   - 또는 **`구글 시트에서 가져오기`** 에 선생님 명단 시트 주소를 붙여넣어도 됩니다.
4. (권장) 같은 화면에서 **비밀번호를 새로 바꿔** 주세요.

### 5단계 — 학생들에게 공유
- 4단계의 **웹앱 URL**을 학생들에게 알려주세요. (QR코드로 만들어 칠판에 띄우면 편해요)
- 학생: **이름 선택 → 학습 → 계획 → 촬영 → 자기평가 → 올리기 → 보드**

🎉 끝! 이제 수업에 쓰면 됩니다.

## 🧒 학생 사용 흐름

1. **입장** — 명단에서 내 이름 누르기
2. **학습** — 애니메이션 원리(잔상·컷·속도)와 플립 애니메이션 4단계 익히기
3. **계획** — 제목·이야기·컷(스토리보드) 짜기
4. **촬영** — 인형/그림을 *조금씩* 움직이며 한 컷씩 찍기
   - 카메라가 켜지면 **지난 컷이 옅게 비쳐(어니언 스킨)** 위치 맞추기가 쉬워요.
   - 컷 순서는 ◀ ▶, 삭제는 ✕, 속도는 슬라이더(🐢↔🐇)로 조절
5. **자기평가** — 확인 목록 4가지를 모두 체크해야 **올리기** 가능 + 스스로 별점
6. **보드** — 친구 작품을 **하트 많은 순**으로 감상, ❤️ 하트 / ⭐ 동료평가 / 💬 댓글

> 1위 작품에는 👑, 1~3위에는 🥇🥈🥉 가 표시됩니다. 보드는 8초마다 자동으로 갱신돼요.

## ❓ 자주 묻는 질문

**Q. 카메라가 안 켜져요.**
A. 기기/환경에 따라 카메라 직접 제어가 막힐 수 있어요. 이때는 **“사진 가져오기”** 단추를 누르면 기기 카메라가 열리거나 사진을 고를 수 있습니다. 기능은 똑같아요.

**Q. 코드를 고쳤어요. 어떻게 반영하나요?**
A. 편집기에서 저장 후 **`배포` → `배포 관리` → (연필✏️ 편집) → 버전을 `새 버전`으로 → `배포`**. (URL은 그대로 유지됩니다.)

**Q. 영상/명단은 어디에 저장되나요?**
A. 선생님 구글 드라이브의 **"애니메이션 스튜디오 데이터"** 시트(명단·하트·댓글·평가)와 **"애니메이션 스튜디오 작품"** 폴더(사진 컷)에 저장됩니다. 직접 열어 확인·백업할 수 있어요.

**Q. 비밀번호를 잊었어요.**
A. 앱스 스크립트 편집기 → **`프로젝트 설정(⚙️)` → `스크립트 속성`** 에서 `TEACHER_PIN` 값을 확인/수정하면 됩니다.

**Q. 몇 명까지 괜찮나요?**
A. 한 학급(약 30명) 규모에 맞춰 설계했습니다. 여러 반이 같은 시간에 몰리면 느려질 수 있어요.

## 🔧 영상 화질·길이 조절 (선택, 고급)

용량을 줄이려고 사진을 작게 저장합니다. 바꾸려면 `App.html` 안의 숫자를 조정하세요.
- `S.fileToFrame(file, 480)` / `S.dataUrlToFrame(...)` 의 **480** → 가로 픽셀(크게=고화질·고용량)
- `toDataURL('image/jpeg', 0.55)` 의 **0.55** → 화질(0~1)

> 너무 크게 하면 저장이 느려지거나 실패할 수 있으니 조금씩 바꿔 테스트하세요.
